Architecture

How It Works

Search — Tavily REST API

  • 5 targeted queries fired in parallel per research call
  • Live results — not limited by model training cutoff
  • Relevance score filter ≥ 0.3 drops low-quality hits
  • Domain blocklist removes known noise sources
  • All results assigned globally sequential [N] indices

Synthesis — Moonshot AI (Kimi k2.5)

  • Single synthesis call per research phase — no tool loops
  • Raw fetch in researcher.ts to preserve Kimi response fields
  • OpenAI SDK for memo generators (structured JSON output)
  • 40,000 char truncation before memo generation
  • JSON code-fence stripping before parse

Full Memo Pipeline

4-Step Research Flow

1
Research — parallel dual track
researchCompany() + analyzeSocialSentiment() run simultaneously via Promise.all. Each fires 5 Tavily queries in parallel, then makes a single Kimi synthesis call. Results include full source URL list.
2
Memo generation
generateMemo() receives merged research + sentiment text, produces a structured InvestmentMemo JSON with 13 fields including companyStatus (ACTIVE / ACQUIRED / IPO / DEFUNCT) which drives conditional rendering.
3
Markdown save
formatMemoAsMarkdown() renders all sections with Markdown formatting, bullet points, tables, and source list. Saved as memo-{slug}-{yyyymmdd}.md.
4
PDF export (optional — --pdf)
Generates a Python script using reportlab, executes it, then deletes the temp file. Auto-installs reportlab if not present. Company status gates which sections render.
Brief Memo Pipeline

2-Step Chinese Research Flow

1
Research — researchCompanyChinese()
Fires 5 Tavily queries in parallel: WeChat public articles, official site anchor query, recent news, funding rounds, founder/team. Uses buildSearchContext() to assign globally sequential [N] indices across all results. Returns { rawContent, sources[] }.
2
Memo generation — generateShortMemo()
Kimi produces a 6-field JSON: headerSnapshot, productAndTechnology, teamBackground, competitiveLandscape, recentDevelopments, analystTake. All facts cited with （来源[N]）; single-source claims flagged （待核实）. Saved as brief-{slug}-{yyyymmdd}.md.

⚠️ npm flag gotcha

Always use -- before arguments. Without it, npm intercepts flags like --brief as its own config options and they never reach the script.

Reliability & Limits

What to Know

🔄

Auto-retry

withRetry() catches rate_limit and engine_overloaded errors, waits 60s, retries indefinitely. All other errors are rethrown immediately.

📡

Live search

Tavily provides current results — not bounded by Kimi's training cutoff. Relevance filtered at score ≥ 0.3 with a domain blocklist for noise reduction.

🗃️

Context limit

Research content truncated at 40,000 chars before memo generation. JSON output is stripped of code fences before parsing.

Known ambiguity issue

Generic or homonymous brand names (e.g. "Wanaka", "Rezona") can attract irrelevant search results. Mitigate with the --name flag alongside a URL to anchor official-site queries.

Source citation integrity

All Tavily URLs are numbered globally before synthesis. The model's inline [N] references map directly to the numbered sources list at the bottom of every memo.
